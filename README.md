# WasteMap ♻️

**Goal:** Help communities coordinate waste pickup and track progress toward sustainable cities (SDG 11 + 12).

## Tech Stack
- MongoDB Atlas  
- Express & Node.js  
- React (Create React App)
- Deployed on Render + Vercel

## Features
- User Register / Login (JWT Auth)
- Residents submit waste pickup requests (auto geolocation)
- Admin dashboard to update status (Pending → In Progress → Completed)
- Map visualization with color‑coded markers
- “My Requests” page for each user
- Live deployment links
